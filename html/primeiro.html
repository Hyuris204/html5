Algoritmo

O que é um algoritmo?

O algoritmo é definido através de cogigos fonte (linguagem), é uma sequencia finita de passos executados em uma ordem especific e detalhada para Realizar uma ação ou gerar um Resultado

Lógica de Programação

A forma como você organiza qual passo você deve dar em cada momento para realizar a ação que deseja. Ou seja, irá aprender os comandos (passos) e vai escrever os codigos (o roteiro) usando a lógica para definir qual comando vai ser executado primeiro, qual vai ser depois e assim or diante até atingir seu objetivo.

Pensar em cada passo que a gente vai dar detalhadamente nos dará uma visão completa de algo que fazemos diariamente de modo automático, sem pensar. Provavelmente a gente vai identificar algo fora desse padrão correto. É Como uma receita.

Algoritmos computacionais

São os passos a serem seguidos por um módulo processador e seus respectivos usuarios que, executados na ordem correta, conseguem realizar determinada tarefa
